# GeoCalc DSL: справка для пользователя

Эта версия документации написана для автора задач, учителя и ученика:
- минимум внутренних деталей;
- максимум практики;
- быстрый поиск нужной конструкции.

## 1. Быстрый старт за 2 минуты

```python
# Базовые точки (их можно перетаскивать в UI)
A = Point(-1.5, -1.5)
B = Point(2.5, -1.5)
C = Point(1.8, 1.5)

# Стороны треугольника
AB = A | B
BC = B | C
AC = A | C

# Пример вычислений
print(abs(AB))          # длина AB
print(angle(B, A, C))   # угол BAC в градусах
```

Идея языка:
1. Создаете точки, прямые, окружности.
2. Комбинируете их операторами (`|`, `&`, `~`, `//`, `%`, `@`).
3. Получаете новые объекты или числа.

## 2. Главные объекты

В DSL есть 4 основных типа:
1. `Point` — точка
2. `Line` — прямая
3. `Circle` — окружность
4. `Vector` — вектор

Плюс функции:
1. `mid`
2. `div`
3. `dist`
4. `angle`
5. `bisect`

## 3. Шпаргалка операторов

### `|`
1. `Point | Point -> Line`  
   Прямая через 2 точки.
2. `Point | Vector -> Line`  
   Прямая через точку в направлении вектора.
3. `Point | number -> Circle`  
   Окружность с центром в точке и радиусом `number`.
4. `Line | Point -> Circle`  
   Окружность через 3 точки (`line.p1`, `line.p2`, `point`).

### `&`
1. `Line & Line -> Point | None`  
   Точка пересечения прямых, или `None` для параллельных.
2. `Line & Point -> Point`  
   Проекция точки на прямую.
3. `Line & Circle -> (Point, Point) | None`  
   Пересечение прямой и окружности.
4. `Circle & Line -> (Point, Point) | None`
5. `Circle & Circle -> (Point, Point) | None`

### `~`
1. `~Line -> Vector`  
   Нормаль прямой.
2. `~Vector -> Vector`  
   Поворот вектора на 90°: `(-dy, dx)`.

### Прочие
1. `Point // Line -> Line`  
   Прямая через точку, параллельная линии.
2. `Point @ Line -> Point`  
   Отражение точки относительно линии.
3. `Point % Circle -> (Line, Line) | None`  
   Касательные из точки к окружности.
4. `Point in Line -> bool`
5. `Point in Circle -> bool`

### Арифметика
1. `Point - Point -> Vector`
2. `Point - Vector -> Point`
3. `Point + Vector -> Point`
4. `Vector + Vector -> Vector`
5. `Vector - Vector -> Vector`
6. `Vector * number -> Vector`
7. `number * Vector -> Vector`
8. `Vector / number -> Vector`
9. `-Vector -> Vector`
10. `abs(Vector) -> float` (длина)
11. `abs(Line) -> float` (длина опорного отрезка линии)
12. `Vector ^ Vector -> float` (разность направлений в градусах)

## 4. Функции DSL

### `mid(A, B) -> Point`
Середина отрезка `AB`.

### `div(A, B, k, n=1.0) -> Point`
Деление отрезка `AB` в отношении `k:n`:
- параметр `t = k / (k + n)`
- точка `A + t*(B - A)`

Важно: `k + n != 0`.

### `dist(P, obj) -> float`
1. `dist(Point, Line)` — расстояние от точки до прямой
2. `dist(Point, Point)` — расстояние между точками

### `angle(B, A, C) -> float`
Угол `BAC` (вершина в `A`) в градусах.

### `bisect(A, B, C, n=1)`
Деление угла `ABC` (вершина в `B`):
1. `n=1` -> одна биссектриса (`Line`)
2. `n=2` -> две линии (внутренняя/внешняя)
3. `n>2` -> `n-1` линий равного деления угла

## 5. Удобные автоправила языка

### 5.1 `A = Point(...)` делает точку базовой
Если пишете:
```python
A = Point(1, 2)
```
то в UI эта точка становится перетаскиваемой.

### 5.2 Неявные линии `AB`, `BA`
Если `AB` не объявлена явно, DSL может автоматически понять ее как `A | B`.

Пример:
```python
A = Point(0, 0)
B = Point(3, 0)
print(abs(AB), AB is BA)   # 3.0 True
```

Работает только для имен из двух букв.

### 5.3 Кортежи в операторе `|`
Внутри `|` кортежи вида `(x, y)` превращаются в точки:
```python
L = (0, 0) | (1, 1)
M = A | (2.5, -1.0)
```

## 6. Cookbook: типовые задачи 7-9 класса

### 6.1 Медианы и центр тяжести
```python
A = Point(-2, -1)
B = Point(3, -1)
C = Point(1, 2)

M = mid(B, C)
N = mid(A, C)

AM = A | M
BN = B | N
G = AM & BN
```

### 6.2 Высоты и ортоцентр
```python
A = Point(-2, -1)
B = Point(3, -1)
C = Point(1, 2)

AB = A | B
AC = A | C

H1 = C | ~AB
H2 = B | ~AC
H = H1 & H2
```

### 6.3 Окружность, описанная около треугольника
```python
A = Point(-2, -1)
B = Point(3, -1)
C = Point(1, 2)

circ = Circle(A, B, C)
```

### 6.4 Касательные из точки к окружности
```python
O = Point(0, 0)
R = 3
c = O | R

P = Point(5, 1)
t1, t2 = P % c
```

### 6.5 Проверка гипотезы
```python
A = Point(-2, -1)
B = Point(3, -1)
C = Point(1, 2)

AB = A | B
BC = B | C

print(dist(A, BC))
print(B in AB)
```

## 7. Скрытие служебных построений

Если имя начинается с `_`, объект не показывается на чертеже (при выключенном `Show intermediates`):
```python
_aux = A | B
```

Полезно для промежуточных линий/точек в решениях.

## 8. Что может вернуть `None`

Проверяйте такие случаи:
1. `Line & Line` для параллельных прямых
2. `Circle & Line` и `Circle & Circle`, если пересечения нет
3. `Point % Circle`, если точка внутри окружности

Пример безопасной проверки:
```python
X = L1 & L2
if X is None:
    print("Нет пересечения")
```

## 9. Частые ошибки и как чинить

1. `TypeError: Operation is not defined for ...`  
Причина: оператор применен к несовместимым типам.  
Решение: свериться с таблицей операторов выше.

2. `GeometryError: A line requires two distinct points`  
Причина: пытаетесь построить прямую по одинаковым точкам.

3. `GeometryError: Circle radius must be positive`  
Причина: радиус `<= 0`.

4. `NameError: name 'AB' is not defined`  
Причина: не сработал неявный алиас (нет точек `A`, `B`, либо имя не двухбуквенное).

## 10. Ограничения текущего DSL

1. Это численная геометрия (не символьная доказательная система).
2. Возможны эффекты точности `float` в граничных случаях.
3. `AB`-алиасы есть только для двухбуквенных имен.

## 11. Разрешенные builtins

Доступны только:
1. `abs`
2. `isinstance`
3. `len`
4. `max`
5. `min`
6. `print`
7. `range`
8. `round`
9. `sum`

Плюс модуль `math` как `math`.
